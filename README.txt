DESCRIPTION

- data_processing folder contains all the scripts for processing the raw dataset include both jupyter notebook and python file
- drawings folder contains the gif file for the drawings
- processed_data folder contains the processed data necessary for the webpage
- src folder contains all image and javascript file necessary for the webpage

INSTALLATION

python package required to run scripts:
- pycountry
- numpy
- pandas

No installation required to run the webpage

EXECUTION

We highly recommend that you use the latest Firefox browser to see the webpage.
Easy way to set up the webpage is using python:
python3 -m http.server 8000
